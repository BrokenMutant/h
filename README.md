# h
h is BrainFuck in disguise.

It's just BrainFuck but uses only "h".

BrainFuck symbols are translated into strings of "h".


* '>' - h
* '<' - hh
* '+' - hhh
* '-' - hhhh
* '.' - hhhhh
* ',' - hhhhhh
* '[' - hhhhhhh
* ']' - hhhhhhhh
